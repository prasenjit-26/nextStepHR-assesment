Task 1:
We're excited to have you move forward in our interview process! Your day task will be to build a simple Todo-List application. You will have 1 day to complete the task. Below, you'll find the specific details and technology stack to use:

## Task Description:
Create a Todo-List app with basic user login functionality. Feel free to use any Large Language Model (LLM) tools or AI assistants you like during development.

## Backend Requirements:
	•	Platform: Supabase
	•	Database: PostgreSQL
	•	Authentication: Supabase Auth (email + password is sufficient)
	•	Authorization: Row Level Security (RLS) to ensure users can only access their own data
	•	Realtime (Optional): Supabase Realtime for live updates
	•	Backend Logic: Supabase Edge Functions (Deno + TypeScript) only if additional server logic is required

## Frontend Requirements:
	•	Framework: React 18 with Vite
	•	Language: TypeScript
	•	Styling: Tailwind CSS + shadcn/ui
	•	Routing: React Router DOM
	•	Server State Management: TanStack React Query
	•	Forms & Validation: React Hook Form + Zod
	•	Backend Communication:
	•	Supabase JavaScript Client
	•	Direct interaction with Supabase Auth, Database, and Realtime APIs

## Deliverables:
- Functional Todo application
- Backend API with generated OpenAPI documentation
- Frontend application utilizing the backend API
- Source code repository link (e.g., GitHub or GitLab)

## Evaluation:
We will primarily evaluate your solution on functionality, structure, clarity of code, and proper integration of the requested technologies.
